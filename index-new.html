Anti-aede

Script para Greasemonkey o Tampermonkey que indica que noticias son de medios de AEDE para facilitar el boicot.

Si encontráis algún bug o tenéis alguna propuesta podéis usar el apartado de Issues o Pull Requests de Github.

Instalación

  • Instalar Greasemonkey para Firefox o Tampermonkey para Chrome/Chromium
  • Instalar el anti-AEDE desde este enlace.

Configuración

  • Una vez instalado, en esta misma url, se verá un panel desde el que configurar las listas de boicots.
  • Los dominios en los que actuará el script, se configuran desde la configuración de tampermonkey/greasemonkey
  • Para modificar las listas de boicots "oficiales" hay que hacer un pull request en el repositorio de anti-AEDE
  • Para utilizar listas particulares, necesitas que sean accesibles mediante alguna url.
